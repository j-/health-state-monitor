    Health state monitor

    Simple state machine which can be used to monitor the health of a system, job, server, routine, connection, user or anything else.

    Usage

    Create and configure a monitor instance. Ping it whenever there is activity from the monitored system. Check the state at any given time or watch it for changes.

    var monitor = new Monitor();
monitor.warningThreshold = 1000 * 60 * 4; // 4 minutes
monitor.dangerThreshold = 1000 * 60 * 5; // 5 minutes

// Monitor any arbitrary system
system.on('message', function () {
    monitor.ping();
});

// Check state periodically
setInterval(function () {
    if (monitor.state !== Monitor.STATES.HEALTHY) {
        // System is no longer healthy
        // Handle appropriately
    }
}, 1000 * 10); // Every 10 seconds
